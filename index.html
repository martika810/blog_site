Learning GIT

GIT is a version control tool . This means it helps you controlling all changes done on a set of files. Additionally you can share those files

How to create a github repository

Introduction

Let's say you have some code in a folder in you laptop and you would like to share it in github. What would you need to do?

Below are the steps you need to follow to create a git repository. This will allow you to share your code with friends, other colleagues, hiring companies, etc

There are several git repositories websites that you can use like BitBucket, Gitlab, Github etc. In this examples, I will use github. The steps are exactly the same of any other

1.Create a github user

Browse to github.com and sign up. It is free, all you need to do is providing a user name, email and password

2.Create a new repository in github

Hover on the profile icon and then click on 'New repository'. Then enter a name for the repo. In this case 'firstRepo'

3.Convert your folder in a git repository

From the terminal, move to the folder where the files you want to publish are cd folder_pathand then execute git init . This will convert your folder in a git repository and generate the master branch
